# dtm-dashboard

Aplicacion web para consultar informacion de la competicion DTM.

La app centraliza datos clave como proximas carreras, eventos por temporada, clasificacion y detalles relacionados con equipos y pilotos.

## Tecnologias de la aplicacion

- Vue 3
- Vite
- TypeScript
- TheSportsDB API (fuente de datos)

## Herramientas de desarrollo

- ESLint
- Prettier
- PicoCSS
